# 帮助和支持 {#intro}

感谢你使用 AirCode，不论遇到任何问题，优先推荐你阅读和搜索「[文档](https://docs.aircode.io)」。我们的团队会将你可能遇到的每个问题整理到文档中，并且不断完善。

2. 我们的 [Discord 讨论区](https://discord.com/invite/XrMVdYdEuY)
